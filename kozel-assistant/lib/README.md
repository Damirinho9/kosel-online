# Библиотеки расширения

Эта папка содержит внешние библиотеки для расширения.

## TensorFlow.js (опционально)

Для работы ML функций нужно скачать TensorFlow.js:

**См. инструкцию:** `../INSTALL_TENSORFLOW.md`

### Быстрая установка:

```bash
# Из папки kozel-assistant/lib/
wget https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.11.0/dist/tf.min.js
```

Или скачайте вручную и поместите сюда файл `tf.min.js`

---

**Без TensorFlow.js:** Расширение работает полностью, но ML функции будут недоступны.
