# Библиотеки TensorFlow.js

## ⚠️ ТРЕБУЕТСЯ РУЧНАЯ УСТАНОВКА

**Файл `tf.min.js` отсутствует или пуст!**

### Как установить:

1. **Откройте в браузере:**
   ```
   https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.11.0/dist/tf.min.js
   ```

2. **Сохраните файл (Ctrl+S):**
   - Имя: `tf.min.js`
   - Папка: эта папка (`kozel-assistant/lib/`)
   - Размер: должен быть **~1.4-1.8 MB**

3. **Перезагрузите расширение:**
   - Откройте `chrome://extensions/`
   - Нажмите кнопку обновления

### Проверка успешной установки:

После перезагрузки расширения откройте консоль (F12) на kozel-online.com:

```
✓ Должны увидеть:
[Background ML] ✓ TensorFlow.js загружен: 4.11.0
[Background ML] ✓ ML модель инициализирована

❌ Если видите:
[Background ML] ⚠️ ML недоступен
```

### Почему локально?

- Chrome Extension Manifest V3 требует локальные файлы
- CSP сайта блокирует загрузку с CDN
- Background service worker не может загружать внешние скрипты

### Что если не установить?

✅ **Расширение работает полностью!**
- Основной AI V2.0 работает (адаптивные стратегии)
- Профилирование игроков работает
- Все рекомендации работают

❌ **Не будет:**
- Нейронная сеть (ML предсказания)
