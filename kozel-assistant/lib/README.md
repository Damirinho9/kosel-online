# Библиотеки TensorFlow.js

## ⚠️ ТРЕБУЕТСЯ РУЧНАЯ УСТАНОВКА

**Файл `tf.min.js` в этой папке - заглушка!**

### Почему нужна ручная установка?

- **Chrome Extension Manifest V3** не разрешает загрузку скриптов с внешних CDN
- Это требование безопасности браузера Chrome
- Автоматическая загрузка невозможна из-за Content Security Policy

### Как установить:

**Способ 1 (через браузер - рекомендуется):**

1. **Откройте в браузере Chrome:**
   ```
   https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.11.0/dist/tf.min.js
   ```

2. **Нажмите Ctrl+S (или Cmd+S на Mac)** для сохранения

3. **ВАЖНО: Сохраните файл КАК ЗАМЕНУ существующего:**
   - Папка: `kozel-assistant/lib/`
   - Имя файла: `tf.min.js` (точно так!)
   - Размер должен быть **~1.4-1.8 MB** (не 2 KB!)

4. **Перезагрузите расширение:**
   - Откройте `chrome://extensions/`
   - Нажмите кнопку обновления

**Способ 2 (альтернативные CDN):**

- https://unpkg.com/@tensorflow/tfjs@4.11.0/dist/tf.min.js
- https://github.com/tensorflow/tfjs/releases/download/tfjs-v4.11.0/tf.min.js

### Проверка успешной установки:

После установки в папке `kozel-assistant/lib/` должен быть файл:
```
lib/
└── tf.min.js  (~1.8 MB, НЕ 2 KB!)
```

После перезагрузки расширения откройте консоль (F12):

```
✓ Должны увидеть:
[ML Offscreen] ✓ TensorFlow.js загружен: 4.11.0
[ML Offscreen] Инициализация ML...

❌ Если видите большой красный блок с инструкциями:
Файл не установлен или это всё ещё заглушка
```
