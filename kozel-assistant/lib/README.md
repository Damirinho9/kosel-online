# Библиотеки TensorFlow.js

## ✅ Автоматическая загрузка с CDN

**TensorFlow.js теперь загружается автоматически!**

### Изменения в версии 2.0.1:

TensorFlow.js загружается напрямую с CDN в offscreen document:
- Не требуется ручная установка
- Библиотека загружается автоматически при первом запуске
- Используется официальный CDN: jsdelivr.net
- Версия: TensorFlow.js 4.11.0

### Как это работает:

- Offscreen document не подвержен CSP ограничениям Service Worker
- TensorFlow.js загружается из `offscreen.html` с CDN
- Загрузка происходит один раз при создании offscreen document
- ML функции доступны сразу после установки расширения

### Проверка успешной загрузки:

После установки расширения откройте консоль (F12) на kozel-online.com:

```
✓ Должны увидеть:
[ML Offscreen] ✓ TensorFlow.js загружен: 4.11.0
[ML Offscreen] Инициализация ML...

❌ Если видите:
[ML Offscreen] ✗ TensorFlow.js не загружен
```
