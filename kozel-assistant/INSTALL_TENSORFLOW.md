# Установка TensorFlow.js для ML функций

## ⚠️ Важно: Ручная установка обязательна

**Файл TensorFlow.js необходимо скачать вручную из браузера.**
CDN может блокировать автоматические запросы.

## Пошаговая инструкция

**Способ 1 (рекомендуется):**

1. **Откройте в браузере Chrome:**
   ```
   https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.11.0/dist/tf.min.js
   ```

2. **Нажмите Ctrl+S (или Cmd+S на Mac)** для сохранения

3. **Выберите папку:** `kozel-assistant/lib/`

4. **Имя файла:** `tf.min.js` (точно так!)

5. **Проверьте размер:** должно быть ~1.4-1.8 MB

**Способ 2 (альтернативный):**

1. Скачайте с альтернативного CDN:
   ```
   https://unpkg.com/@tensorflow/tfjs@4.11.0/dist/tf.min.js
   ```

2. Или напрямую с GitHub:
   ```
   https://github.com/tensorflow/tfjs/releases/download/tfjs-v4.11.0/tf.min.js
   ```

### Проверка установки

После установки в папке `kozel-assistant/lib/` должен быть файл:
```
lib/
└── tf.min.js  (~1.8 MB)
```

## Что если не установить?

Если файл отсутствует:
- ❌ ML функции будут недоступны
- ✅ Основной AI V2.0 будет работать полностью (адаптивные стратегии)
- ✅ Профилирование игроков будет работать
- ✅ Все остальные функции доступны

## После установки

1. Перезагрузите расширение в Chrome:
   - `chrome://extensions/`
   - Нажмите кнопку обновления расширения

2. Откройте kozel-online.com

3. Проверьте консоль (F12):
   ```
   [ML Loader] ✓ TensorFlow.js загружен: 4.11.0
   [Козёл Помощник ML] ✓ ML модель загружена
   ```

## Размер файла

- TensorFlow.js: ~1.8 MB
- Общий размер расширения: ~2.3 MB

---

**Важно:** TensorFlow.js нужен только для ML функций (нейронная сеть). Основной AI отлично работает без него!
