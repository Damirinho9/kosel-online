# Установка TensorFlow.js для ML функций

## ✅ Автоматическая установка (версия 2.0.1+)

**TensorFlow.js загружается автоматически с CDN!**

С версии 2.0.1 больше не требуется ручная установка.

## Как это работает:

- TensorFlow.js загружается автоматически при первом запуске расширения
- Используется offscreen document для обхода CSP ограничений
- Библиотека загружается с официального CDN: jsdelivr.net
- Версия: TensorFlow.js 4.11.0

## Проверка работы ML

1. Установите расширение
2. Откройте kozel-online.com
3. Проверьте консоль (F12):
   ```
   ✓ Должны увидеть:
   [ML Offscreen] ✓ TensorFlow.js загружен: 4.11.0
   [ML Offscreen] Инициализация ML...
   ```

## Что если CDN недоступен?

Если CDN заблокирован или недоступен:
- ❌ ML функции будут недоступны
- ✅ Основной AI V2.0 будет работать полностью (адаптивные стратегии)
- ✅ Профилирование игроков будет работать
- ✅ Все остальные функции доступны

В этом случае расширение продолжит работать в полном объеме, за исключением ML предсказаний.

## Старая версия (до 2.0.1)

Если вы используете версию до 2.0.1:

1. **Откройте в браузере Chrome:**
   ```
   https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.11.0/dist/tf.min.js
   ```

2. **Нажмите Ctrl+S (или Cmd+S на Mac)** для сохранения

3. **Выберите папку:** `kozel-assistant/lib/`

4. **Имя файла:** `tf.min.js` (точно так!)

5. **Проверьте размер:** должно быть ~1.4-1.8 MB

---

**Важно:** TensorFlow.js нужен только для ML функций (нейронная сеть). Основной AI отлично работает без него!
